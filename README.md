# AWS Cloud Practitioner — Hands-On Labs

A collection of hands-on AWS labs I completed while preparing for the AWS Certified Cloud Practitioner (CCP) certification. Each lab goes beyond the multiple-choice exam scope to build real, practical experience with core AWS services within the Free Tier.

## Labs

| # | Lab | Key services |
|---|-----|--------------|
| 1 | [VPC and Networking](./labs/lab-01-vpc-networking) | VPC, Subnets, IGW, Route Tables, EC2 |
| 2 | [EC2 and Security Groups](./labs/lab-02-ec2-security-groups) | EC2, Security Groups, SSH, Apache |
| 3 | [S3 Storage](./labs/lab-03-s3-storage) | S3, Versioning, Lifecycle Policies |
| 4 | [IAM](./labs/lab-04-iam) | IAM, MFA, Policies, Roles |
| 5 | [RDS Managed Database](./labs/lab-05-rds) | RDS, Multi-AZ, Backups |
| 6 | [Lambda Serverless](./labs/lab-06-lambda) | Lambda, CloudWatch Logs |
| 7 | [CloudWatch Monitoring](./labs/lab-07-cloudwatch) | CloudWatch Metrics, Alarms, Dashboards |
| 8 | [Cost Management](./labs/lab-08-cost-management) | Cost Explorer, Budgets, Pricing Calculator |
